# ✈️ IndiGo Airline Data Analysis (Power BI Project)

## 📊 Project Overview
This project analyzes **IndiGo Airlines data** to uncover insights related to revenue, passenger trends, operational performance, and pricing strategy. The dashboard is built using **Power BI** with raw CSV/Excel data.

---

## 🎯 Problem Statement
The airline industry operates in a highly competitive and dynamic environment where efficient utilization of resources, pricing strategies, and operational performance are critical.

This project aims to:
- Analyze airline revenue trends  
- Understand passenger demand  
- Evaluate delays and cancellations  
- Identify key routes and hubs  

---

## 🛠️ Tools & Technologies
- Power BI  
- Microsoft Excel / CSV  
- DAX  

---

## 📈 Key KPIs
- 💰 Total Revenue: ₹2913.61 Cr  
- 👥 Total Passengers: 580K  
- 📊 Avg Load Factor: 0.64  
- ⏱️ On-Time %: 48%  
- ❌ Cancellation %: 2%  
- 💸 Avg Ticket Price: ₹5.03K  
- ⏳ Avg Delay: 31.79 minutes  

---

## 🔍 Key Insights

### 📌 Revenue
- Highest revenue month: **September (~₹252 Cr)**
- Top route: **Mumbai → Ahmedabad (~₹66.55 Cr)**
- Revenue fluctuates across the year

### 👥 Passengers
- Total: **580K passengers**
- Busiest route: **Pune → Delhi (~11K)**
- Seasonal travel trends observed

### ✈️ Operations
- Delayed flights: **50%**
- On-time flights: **48%**
- Cancelled: **2%**
- Delays increased after July

### 📊 Load Factor
- Average: **0.64**
- Highest: **Hyderabad → Ahmedabad (~0.69)**
- Lowest: Bangalore routes (~0.61)

### 💸 Pricing
- Avg Ticket Price: **₹5.03K**
- Highest priced route: **Pune → Delhi**
- High price ≠ always high revenue

### 🌍 Network
- Major hubs:
  - Delhi  
  - Mumbai  
  - Bangalore  
  - Hyderabad
